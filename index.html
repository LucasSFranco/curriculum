Plataforma de academia

Meu primeiro website, elaborado para um projeto escolar e desenvolvido inteiramente com HTML, CSS, JavaScript e Bootstrap. Trata-se de uma plataforma para academia com venda de produtos, com três níveis de acesso:
  • administrador: pode listar, excluir e editar funcionários, clientes e produtos, atualizar o histórico de evolução dos clientes e ver o relatório de vendas;
  • funcionário: pode realizar as mesmas ações que o administrador com relação aos clientes; e,
  • cliente: pode ver os produtos disponíveis, as notificações de ausência em treinos e de mensalidades atrasadas e o seu histórico de evolução.

App de planejamento

Aplicativo elaborado com Imba.js cujo funcionamento consiste em adicionar as tarefas diárias a uma lista e, após concluí-las, adicionar o horário em que cada tarefa foi realizada utilizando uma legenda colorida. Todos esses dados eram então armazenados localmente por meio do Dexie.js.

E-commerce

Plataforma de vendas elaborada para um projeto escolar em PHP, HTML, CSS e JavaScript utilizando um banco MySQL. Utiliza três níveis de acesso:
  • administrador: pode listar, adicionar, excluir e editar produtos e categorias de produtos;
  • cliente: pode ver os produtos disponíveis na loja, adicioná-los ao carrinho de compras e finalizar a compra utilizando a API de desenvolvimento do PagSeguro; e,
  • visitante: pode se cadastrar, realizar login e ver os produtos disponíveis na loja.
